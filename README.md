ИНСТРУКЦИЯ ПО УСТАНОВКЕ НОВГО БЛОКА:

1) скопировать папку "projects_module" и расположить ее по пути:  /modules/custom/projects_module/

2)  активировать модуль в админке Drupal:
   1. найти вкладку "Расширения"
   2. найти "Projects Module"
   3. поставить галочку, нажать "Установить"

3) создать тип материала "Проект" (машинное имя: proekt) (Структура -> Типы материалов -> Добавить тип материала)
   1. создать три  поля: описание (field_opisanie), изображение (field_izobrazhenie), дата окончания(field_data_okonchaniya).

4) расположить блок на главной странице:
   1. Структура -> Схема блоков
   2. выбрать область расположения и нажатть кнопку "Расположить"
   3. найти блок "Последние проекты"

5) создать немкольео проектов, блок выведет последние 3 опубликованные.






АРХИТЕКТУРНЫЕ РЕШЕНИЯ

1) разделение функций
    1. CSS в отдельном файле, стили вынесены в "/css/projects-styles.css"
    2. PHP код отвечает только за вывод данных, CSS отвечает только за визуал
    3. легче вносить изменения

2) современная Drupal-архитектура
    1. плагин блока (BlockBase), из-за совместимость с Drupal 10, и его актуальности
    2. Entity Query API, для безопасных запросов без SQL
    3. использование file_url_generator для URL изображений, т.к. генерирует абсолютный путь

3) проверки на пустые поля
    1. защита от ошибок если поле не существует или оно пустое

4) структурированные классы (приближенные к БЭМ)
